What dobbe does

Vulnerability Resolution

Scan Dependabot alerts, triage by risk, upgrade packages, run tests, retry on failure, create PR. Fully automated.

/dobbe-vuln-resolve

Code Review

Fetch open PRs, perform deep AI review with risk assessment, post comments directly to GitHub.

/dobbe-review-post

Test Generation

Find coverage gaps, generate tests, verify they pass, retry if not, create PR. Up to 3 iterations.

/dobbe-test-gen

DORA Metrics

Deploy frequency, lead time, change failure rate, MTTR. Computed from your GitHub data.

/dobbe-metrics-dora

Incident Triage

Fetch Sentry issues, AI-powered root cause analysis, severity assessment, resolution suggestions.

/dobbe-incident-triage

Security Audit

Vulnerabilities, licenses, secrets, code quality. Single command, comprehensive report.

/dobbe-audit-report

How it works

1

Install in 10 seconds

Run npx dobbe install and restart Claude Code. Skills appear as /dobbe-* slash commands.

2

Run a command

Type /dobbe-vuln-resolve in Claude Code. The MCP server starts a state machine pipeline and returns step-by-step instructions.

3

Claude does the work

Claude follows instructions, submits results at each step. If tests fail, the server automatically retries with feedback. You get a PR at the end.
